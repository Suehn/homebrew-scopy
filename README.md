# Homebrew Cask for Scopy

Homebrew tap for [Scopy](https://github.com/Suehn/Scopy) - a native macOS clipboard manager.

## Installation

```bash
brew tap Suehn/scopy
brew install --cask scopy
```

## Note

This tap is automatically synced from the main [Scopy repository](https://github.com/Suehn/Scopy/tree/main/Casks).
